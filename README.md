# Budget Calculator

A simple and responsive budgeting web application that helps users track their income and expenses. The application provides a clear overview of spending habits with interactive charts and automatically saves data in the browser using localStorage.

# Preview

# Features

- Add monthly or yearly income
- Record expenses with categories
- View monthly and yearly spending
- Interactive spending breakdown using Chart.js
- Calculate total income, total expenses, and remaining balance
- Remove individual expenses
- Clear all saved data
- Automatically saves data using browser localStorage
- Responsive design for desktop and mobile devices

# Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

# How to Run

1. Clone the repository

```bash
git clone https://github.com/Sophany0354/budget-calculator.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No installation or additional dependencies are required.

# Data Storage

This application uses **browser localStorage** to save user data.

Your income and expenses remain available even after closing and reopening the browser, provided you use the same browser and device.
